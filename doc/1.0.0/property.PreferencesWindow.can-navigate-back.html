property can-navigate-back: gboolean [ read, write ]

Description [src]

Whether gestures and shortcuts for closing subpages are enabled.

The supported gestures are: - One-finger swipe on touchscreens - Horizontal scrolling on touchpads (usually two-finger swipe) - Back mouse button

The keyboard back key is also supported, as well as the Alt+‚Üê shortcut.
